# CalculatorApp

### 개발 환경 및 언어
Android-Studio(java)


### 구현 내용
* 입출력
* prefix -> postfix를 통한 계산
* 일반 사칙 연산
* 괄호 연산
* clear, delete 
* 제곱,세제곱,N승,루트,팩토리얼,pi,부호변환, 1/x
